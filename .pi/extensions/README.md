# Pi Coding Agent Extensions

This directory contains extensions for the Pi Coding Agent platform.

Extensions here enable Pi Coding Agent to use tGD slash commands (e.g., `/tgd-plan`, `/tgd-define`) natively.

## Adding Extensions

Place TypeScript extension files (`.ts`) in this directory. Pi Coding Agent auto-discovers extensions from `.pi/extensions/`.

See the [Pi documentation](https://docs.pi.dev) for extension authoring guidelines.
